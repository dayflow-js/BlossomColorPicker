# Vanilla JavaScript

독립 코어 패키지를 사용하면 `BlossomColorPicker`와 `ChromePicker`를 프레임워크 없이 직접 사용할 수 있습니다.

## 코어 패키지 설치

<PackageTabs pkg="@dayflow/blossom-color-picker" />

## 공유 스타일 가져오기

래퍼 패키지는 공유 스타일을 자동으로 가져오지만, 독립 코어 패키지는 직접 가져와야 합니다.

```ts
import '@dayflow/blossom-color-picker/styles.css';
```

## 기본 사용법

```js
import { BlossomColorPicker } from '@dayflow/blossom-color-picker';
import '@dayflow/blossom-color-picker/styles.css';

const root = document.getElementById('picker');

const picker = new BlossomColorPicker(root, {
  onChange: (color) => {
    console.log('Selected color:', color.hex);
  },
});
```

## 런타임 제어

| 메서드 | 설명 |
| :--- | :--- |
| `setValue(value)` | 현재 값을 코드에서 직접 설정합니다. |
| `getValue()` | 현재 색상을 `BlossomColorPickerColor`로 읽어옵니다. |
| `expand()` | 피커를 펼칩니다. |
| `collapse()` | 피커를 접습니다. |
| `toggle()` | 펼침/접힘 상태를 전환합니다. |
| `setOptions(options)` | 초기화 후 옵션을 업데이트합니다. |
| `destroy()` | 생성된 DOM과 이벤트 리스너를 정리합니다. |

## 런타임 예제

```js
import { BlossomColorPicker } from '@dayflow/blossom-color-picker';

const picker = new BlossomColorPicker(document.getElementById('picker'));

picker.expand();
picker.setValue({
  hue: 200,
  saturation: 50,
  alpha: 100,
  layer: 'outer',
});

picker.setOptions({
  disabled: true,
  openOnHover: false,
});
```

## 래퍼 없이 `ChromePicker` 사용하기

```js
import { ChromePicker } from '@dayflow/blossom-color-picker';
import '@dayflow/blossom-color-picker/styles.css';

const picker = new ChromePicker(document.getElementById('picker'), {
  onChange: (color) => console.log(color.rgba),
});
```
